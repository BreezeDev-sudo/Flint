🔥 Language Reference

The Flint
Programming Language

A lightweight scripting language for CLI tools and real-time graphical applications. Files use the .flt extension. Write once, run with flint.

v0.1 .flt files native windows keyboard + mouse TypeScript core
01

Installation

Drop the installer script into your Flint source folder and run it once. It installs Deno if needed and gives you a standalone flint command.

Windows

powershell
powershell -ExecutionPolicy Bypass -File install.ps1

Mac / Linux

bash
bash install.sh
What the installer does: checks for Deno (installs if missing), compiles flint.exe / flint, copies it to a permanent location, adds it to your PATH.

File layout

text
Flint/
├── main.ts
├── lexer.ts
├── parser.ts
├── interpreter.ts
├── ast.ts
├── install.ps1     ← Windows
└── install.sh      ← Mac / Linux

03

Program Structure

ignite is always first, ash always last. Everything else goes in between.

flint
// 1. Set the run mode (required)
ignite window_app;

// 2. Import modules (optional)
kindle ui;

// 3. Variables, functions
let title of string = "My App";

// 4. Create a window (graphical modes only)
spark window(title, 800, 600);

// 5. Render / game loop
flame {
    clear("#000000");
    // draw and update here...
}

// 6. End (required)
ash;

ignite — Run Modes

ignite must be the first statement. It sets what kind of program you're building.

syntax
ignite MODE;
cli
Terminal / command-line programs. No window.
window_app
Standard windowed GUI application.
fullscreen_app
Fullscreen GUI application.
game
Game with a real-time render loop.
renderer
Custom graphics / rendering tool.
headless
Background process, no window.

Variables

Declared with let, typed with of, assigned with =. After declaration, reassign with just name = value;.

syntax
let NAME of TYPE = VALUE;
TypeDescriptionExample
intInteger numberlet score of int = 0;
floatDecimal numberlet speed of float = 1.5;
stringTextlet name of string = "Flint";
boolTrue or falselet active of bool = true;
voidNo value (functions only)—
flint
let x      of int    = 100;
let speed  of float  = 2.5;
let name   of string = "Player";
let alive  of bool   = true;

// Reassignment
x = x + 10;
name = "Hero";
07

Functions

Declared with func. Parameters use name of type. Add returns TYPE; after the closing brace if the function returns a value.

No return value

flint
func greet(username of string) {
    print("Hello, " + username);
}

With return value

flint
func add(a of int, b of int) {
    return a + b;
} returns int;

let total of int = add(3, 5);   // 8
Note: returns TYPE; goes after the closing }, not inside the function body.
08

Control Flow

Flint has if / else and while. No parentheses around conditions — just write the expression directly.

if / else

syntax
if CONDITION {
    // runs when condition is non-zero / non-empty
} else {
    // optional else block
}
flint
let score of int = 42;

if score > 50 {
    print("You win!");
} else {
    print("Keep trying.");
}

// Equality and comparison operators
if score == 100 { print("Perfect!"); }
if score != 0   { print("Not zero."); }
if score >= 40  { print("Almost there."); }

while

flint
let i of int = 0;

while i < 5 {
    print(to_string(i));
    i = i + 1;
}

Truthy values

Conditions are truthy when they are a non-zero number or a non-empty string. Zero and "" are falsy. Since true compiles to 1 and false to 0, you can use them directly:

flint
let alive of bool = true;
if alive { print("Still going!"); }

// key_down returns 1 (true) or 0 (false) — use directly in if
if key_down("ArrowLeft") { x = x - 4; }

flame — The Main Loop

The flame block runs every frame. Variables declared inside it persist between frames — perfect for tracking player position, score, etc.

Always call clear() first to erase the previous frame before drawing the new one.
flint
let px of int = 100;

flame {
    clear("#111122");

    // Move player on key press
    if key_down("ArrowRight") { px = px + 4; }
    if key_down("ArrowLeft")  { px = px - 4; }

    draw_rect(px, 300, 40, 40, "#ff6b1a");
}

Built-ins — Input

All input functions are called inside flame { }. They return 1 (true) or 0 (false) so you can use them directly in if conditions.

FunctionSignatureDescription
key_downkey_down(key) → intReturns 1 every frame the key is held down.
key_pressedkey_pressed(key) → intReturns 1 only on the first frame the key is pressed. Great for jumps, actions.
mouse_xmouse_x() → intCurrent mouse X position in the window.
mouse_ymouse_y() → intCurrent mouse Y position in the window.
mouse_downmouse_down() → intReturns 1 every frame the mouse button is held.
mouse_pressedmouse_pressed() → intReturns 1 only on the first frame the button is clicked.

Key names

Key names follow the standard DOM KeyboardEvent.key spec. Common ones:

Space bar" " (a single space)
Letters"a" "b" … "z" (lowercase)
Shift, Ctrl, Alt"Shift" "Control" "Alt"

Keyboard movement

flint
let px of int = 400;
let py of int = 300;

flame {
    clear("#000000");

    if key_down("ArrowRight") { px = px + 4; }
    if key_down("ArrowLeft")  { px = px - 4; }
    if key_down("ArrowUp")    { py = py - 4; }
    if key_down("ArrowDown")  { py = py + 4; }

    draw_circle(px, py, 20, "#ff6b1a");
}

Mouse drawing

flint
flame {
    if mouse_down() {
        let mx of int = mouse_x();
        let my of int = mouse_y();
        draw_circle(mx, my, 10, "#ffaa33");
    }
}

// key_pressed — fires only on the frame the key goes down
let jumped of int = 0;
flame {
    if key_pressed(" ") { jumped = 1; }
}

Example — CLI Program

flint
ignite cli;

func fizzbuzz(n of int) {
    let i of int = 1;
    while i <= n {
        if i % 15 == 0 { print("FizzBuzz"); }
        if i % 3  == 0 { print("Fizz"); }
        if i % 5  == 0 { print("Buzz"); }
        i = i + 1;
    }
}

fizzbuzz(20);

Example — Interactive Game

A complete moving-player game using keyboard input, score tracking, and a HUD.

flint
ignite game;

spark window("My Game", 800, 600);

// Player state (persists across frames)
let px    of int = 400;
let py    of int = 300;
let score of int = 0;
let speed of int = 4;

func draw_hud() {
    draw_text("Score: " + to_string(score), 10, 10, 18, "#ffffff");
    draw_text("Arrow keys to move", 10, 34, 13, "#666688");
}

flame {
    clear("#111122");

    // Movement
    if key_down("ArrowRight") { px = px + speed; }
    if key_down("ArrowLeft")  { px = px - speed; }
    if key_down("ArrowUp")    { py = py - speed; }
    if key_down("ArrowDown")  { py = py + speed; }

    // Score on space
    if key_pressed(" ") { score = score + 10; }

    // Clamp to window bounds
    px = max(0, min(px, 768));
    py = max(0, min(py, 568));

    // Draw player
    draw_rect(px, py, 32, 32, "#ff6b1a");
    draw_rect_outline(px, py, 32, 32, "#ffffff");

    draw_hud();
}

ash;
