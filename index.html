The Flint
Programming Language

A lightweight scripting language for CLI tools and real-time graphical applications. Files use the .flt extension. Write once, run with flint.

v0.1 .flt files native windows keyboard + mouse TypeScript core
01

Installation

Drop the installer script into your Flint source folder and run it once. It installs Deno if needed and gives you a standalone flint command.

Windows

powershell
powershell -ExecutionPolicy Bypass -File install.ps1

Mac / Linux

bash
bash install.sh
What the installer does: checks for Deno (installs if missing), compiles flint.exe / flint, copies it to a permanent location, adds it to your PATH.

File layout

text
Flint/
├── main.ts
├── lexer.ts
├── parser.ts
├── interpreter.ts
├── ast.ts
├── install.ps1     ← Windows
└── install.sh      ← Mac / Linux

03

Program Structure

ignite is always first, ash always last. Everything else goes in between.

flint
// 1. Set the run mode (required)
ignite window_app;

// 2. Import modules (optional)
kindle math;

// 3. Variables, structs, functions
let title of string = "My App";

// 4. Create a window (graphical modes only)
spark window(title, 800, 600);

// 5. Render / game loop
flame {
    clear("#000000");
    // draw and update here...
}

// 6. End (required)
ash;

syntax

Game with a real-time render loop.

Custom graphics / rendering tool.
headless
Background process, no window.

Variables

Declared with let, typed with of, assigned with =. After declaration, reassign with just name = value;.

syntax
let NAME of TYPE = VALUE;
TypeDescriptionExample
intInteger numberlet score of int = 0;
floatDecimal numberlet speed of float = 1.5;
stringTextlet name of string = "Flint";
boolTrue or falselet active of bool = true;
int[]Array of integerslet nums of int[] = [1, 2, 3];
string[]Array of stringslet tags of string[] = ["a", "b"];
voidNo value (functions only)—
flint
let x      of int    = 100;
let speed  of float  = 2.5;
let name   of string = "Player";
let alive  of bool   = true;

// Reassignment
x = x + 10;
name = "Hero";
07

Functions

Declared with func. Parameters use name of type. Add returns TYPE; after the closing brace if the function returns a value.

No return value

flint
func greet(username of string) {
    print("Hello, " + username);
}

With return value

flint
func add(a of int, b of int) {
    return a + b;
} returns int;

let total of int = add(3, 5);   // 8

Array parameters

Pass arrays to functions using the type[] parameter type.

flint
func sum_all(nums of int[]) {
    let total of int = 0;
    let i of int = 0;
    while i < len(nums) {
        total = total + nums[i];
        i = i + 1;
    }
    return total;
} returns int;
Note: returns TYPE; goes after the closing }, not inside the function body.
08

Control Flow — if / while

Flint has if / else and while. No parentheses around conditions — just write the expression directly.

if / else

syntax
if CONDITION {
    // runs when condition is non-zero / non-empty / non-empty-array
} else {
    // optional else block
}
flint
let score of int = 42;

if score > 50 {
    print("You win!");
} else {
    print("Keep trying.");
}

if score == 100 { print("Perfect!"); }
if score != 0   { print("Not zero."); }
if score >= 40  { print("Almost there."); }

while

flint
let i of int = 0;

while i < 5 {
    print(to_string(i));
    i = i + 1;
}

Truthy values

A condition is truthy when it is a non-zero number, a non-empty string, or a non-empty array. Zero, "", and empty arrays are falsy. Since true compiles to 1 and false to 0, you can use them directly:

flint
let alive of bool = true;
if alive { print("Still going!"); }

// key_down returns 1 (true) or 0 (false) — use directly in if
if key_down("ArrowLeft") { x = x - 4; }

// Non-empty array is truthy
let items of int[] = [1, 2, 3];
if items { print("Has items"); }
09

for Loop

The for loop counts a variable from a start value up to (but not including) an end value. An optional step controls how much the counter changes each iteration — default is 1. Use a negative step to count downwards.

syntax
for VAR from START to END {
    // VAR goes START, START+1, … END-1
}

for VAR from START to END step STEP {
    // custom increment
}
Exclusive upper bound: the loop variable never reaches END. To iterate indices 0 through n-1, write for i from 0 to n.
flint
// Print 0 through 4
for i from 0 to 5 {
    print(to_string(i));
}

// Step by 2: 0, 2, 4, 6, 8
for n from 0 to 10 step 2 {
    print(to_string(n));
}

// Count down: 5, 4, 3, 2, 1
for k from 5 to 0 step -1 {
    print(to_string(k));
}

// Iterate over an array by index
let scores of int[] = [10, 20, 30];
for i from 0 to len(scores) {
    print(to_string(scores[i]));
}
10

break

break immediately exits the nearest enclosing while or for loop.

flint
// Find first even number in a list
let nums of int[] = [1, 3, 4, 7, 8];
let found of int = -1;

for i from 0 to len(nums) {
    if nums[i] % 2 == 0 {
        found = nums[i];
        break;
    }
}

print(to_string(found));   // 4

12

Logical Operators

and, or, and not combine boolean conditions. They short-circuit: and stops if the left side is falsy; or stops if the left side is truthy.

flint
let alive  of bool = true;
let health of int  = 80;
let mana   of int  = 0;

// and — both must be true
if alive and health > 0 {
    print("Can act");
}

// or — either can be true
if health > 50 or mana > 50 {
    print("Has resources");
}

// not — inverts truthy/falsy
if not alive {
    print("Game over");
}

// Compound conditions
if (health > 0) and (not mana == 0) {
    print("Can cast spell");
}
13

Arrays

Arrays are ordered lists declared with a type[] annotation. Elements are accessed and assigned with [index] — zero-based. You can also index into a string to get a single character.

Declaration & literals

flint
let scores of int[]    = [10, 20, 30];
let names  of string[] = ["Alice", "Bob"];
let empty  of int[]    = [];          // start empty

Reading & writing

flint
let first of int = scores[0];   // 10
scores[1] = 99;                  // update index 1

// String indexing — returns single character
let word of string = "hello";
let ch   of string = word[0];    // "h"

Looping over an array

flint
for i from 0 to len(scores) {
    print(to_string(scores[i]));
}
See also: the Array Functions section for push, pop, len, remove, insert, and concat.
14

Structs

Structs group related fields under a named type. Declare with struct, then create instances anywhere using the struct's name followed by { field = value }. Access or mutate fields with dot notation.

Declaration

syntax
struct NAME {
    field1 of TYPE;
    field2 of TYPE;
}

Instantiation & access

flint
struct Vec2 {
    x of float;
    y of float;
}

let pos of Vec2 = Vec2 { x = 100.0, y = 200.0 };

print(to_string(pos.x));   // 100
pos.x = pos.x + 5.0;        // mutate field

Structs in functions

flint
struct Rect {
    x of int; y of int;
    w of int; h of int;
}

func draw_box(r of Rect, col of string) {
    draw_rect(r.x, r.y, r.w, r.h, col);
}

let wall of Rect = Rect { x = 0, y = 0, w = 800, h = 20 };
draw_box(wall, "#ffffff");

Array of structs

flint
let bullets of Vec2[] = [];
push(bullets, Vec2 { x = 400.0, y = 300.0 });

// Access field of struct in array
let bx of float = bullets[0].x;

spark — Creating a Window

Opens the application window. Call once before the flame block. Only needed in graphical modes.

flint
spark window("My App", 800, 600);
window_width() / window_height(): call these inside flame to get the current window dimensions at runtime — useful for clamping positions or centering UI elements.
16

flame — The Main Loop

The flame block runs every frame. Variables declared outside the flame block persist between frames — use them for game state. Variables declared inside are reset each frame.

Always call clear() first to erase the previous frame before drawing the new one.
flint
let px of int = 100;  // persists across frames

flame {
    clear("#111122");

    if key_down("ArrowRight") { px = px + 4; }
    if key_down("ArrowLeft")  { px = px - 4; }

    draw_rect(px, 300, 40, 40, "#ff6b1a");

    // Delta time for frame-rate independent movement
    let dt of float = delta();
}

These functions mutate arrays in place (except concat and len, which return a value).

Example — Interactive Game

A complete moving-player game using keyboard input, score tracking, delta time, structs, and arrays.

flint
ignite game;
kindle math;

spark window("My Game", 800, 600);

struct Coin {
    x of int;
    y of int;
    alive of int;
}

let px    of float  = 400.0;
let py    of float  = 300.0;
let score of int    = 0;
let speed of float  = 200.0;  // px / sec

let coins of Coin[] = [];
for i from 0 to 5 {
    push(coins, Coin { x = random_int(50, 750), y = random_int(50, 550), alive = 1 });
}

func draw_hud() {
    draw_text("Score: " + to_string(score), 10, 10, 18, "#ffffff");
    draw_text("Arrow keys to move", 10, 34, 13, "#666688");
}

flame {
    clear("#111122");
    let dt of float = delta();

    // Movement
    if key_down("ArrowRight") { px = px + speed * dt; }
    if key_down("ArrowLeft")  { px = px - speed * dt; }
    if key_down("ArrowUp")    { py = py - speed * dt; }
    if key_down("ArrowDown")  { py = py + speed * dt; }

    // Clamp
    px = max(16.0, min(px, 784.0));
    py = max(16.0, min(py, 584.0));

    // Coin collection
    for i from 0 to len(coins) {
        if coins[i].alive {
            let dx   of float = px - coins[i].x;
            let dy   of float = py - coins[i].y;
            let dist of float = sqrt(dx*dx + dy*dy);
            if dist < 24 {
                coins[i].alive = 0;
                score = score + 10;
            }
            draw_circle(coins[i].x, coins[i].y, 10, "#ffcc00");
        }
    }

    // Draw player
    draw_rect(px - 16, py - 16, 32, 32, "#ff6b1a");
    draw_rect_outline(px - 16, py - 16, 32, 32, "#ffffff");

    draw_hud();
}
